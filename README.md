# Linux Configs

Repository to store my Linux configurations and make it easy to set up new machines.

## Structure

- `bash/` - Bash terminal configurations
- `ghostty/` - Ghostty terminal configuration
- `hyprland/` - Hyprland window manager configuration
- `waybar/` - Waybar status bar configuration  
- `zellij/` - Zellij terminal multiplexer configuration

## Notes

- Some configs need additional dependencies
- Monitor names and devices may need adjustments
